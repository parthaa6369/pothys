# Service 4: Portfolio Service

User holdings management, live price feeds, portfolio analytics, and tax calculations.

## Features

- Real-time gold/silver price management
- User holdings tracking
- Portfolio value calculation
- Historical price data (OHLC charts)
- Daily portfolio snapshots
- Performance analytics (gain/loss)
- Tax calculation for capital gains
- Price alert notifications

## Tech Stack

- NestJS 10+
- TypeORM + PostgreSQL
- Redis for price caching
- Scheduled jobs for price updates
- WebSocket for real-time price feeds

## Database

- **Database Name**: `portfolio`
