# Service 5: CMS & Engagement Service

Notification management, admin dashboard, analytics, and system monitoring.

## Features

- Multi-channel notifications (Push, SMS, Email)
- Notification templates with multi-language support
- Admin user management with role-based access
- Admin activity audit logs
- Analytics caching and metrics
- System health monitoring
- Real-time dashboard metrics

## Tech Stack

- NestJS 10+
- TypeORM + PostgreSQL
- Firebase Cloud Messaging (Push)
- Twilio (SMS)
- SendGrid (Email)
- Redis for caching
- RabbitMQ for event handling
